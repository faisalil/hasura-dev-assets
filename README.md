# Deprecated Repository

This repository is deprecated and archived for historical reference.

Do not use this repo for current installs or runtime downloads.

Use the single distribution repository instead:

- https://github.com/faisalil/hasura-dev-cli

- CLI release binaries (`hasura-dev-*`)
- Engine assets (`manifest.json`, `graphql-engine-*`)
- Homebrew formula (`Formula/hasura-dev.rb`)
- Release workflows/scripts
